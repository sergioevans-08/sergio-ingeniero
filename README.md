# Sergio el Ingeniero - Página Web

Una página web moderna, elegante y motivadora diseñada con HTML5, CSS3 y JavaScript vanilla.

## 🎯 Características

- ✨ Diseño moderno en modo oscuro
- 🎵 Sección de música con enlace a tu canal de YouTube
- 💬 Galería de frases motivadoras inspiradoras
- 🎬 Vídeos embebidos de YouTube
- 📱 Totalmente responsive (móvil, tablet, desktop)
- ⚡ Animaciones suaves y transiciones elegantes
- 🚀 Rendimiento optimizado

## 🛠️ Tecnologías

- HTML5 semántico
- CSS3 con gradientes y animaciones
- JavaScript vanilla (sin dependencias)
- Google Fonts (Poppins)
- Responsive design
